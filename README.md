## 🇹🇷 Türkçe

### Proje Hakkında

**Yapay Zekâ Voltranı**, yapay zekânın temel kavramlarını eğlenceli, anlaşılır, etkileşimli ve uygulama temelli bir öğrenme deneyimiyle sunmak amacıyla geliştirilmiştir.

Kullanıcılar farklı yapay zekâ modüllerini temsil eden aslanları eğitir, laboratuvar görevlerini tamamlar, pilot sınavlarını geçer ve tüm bileşenleri birleştirerek kendi Yapay Zekâ Voltranlarını oluşturur.

Proje yalnızca bilgi sunan statik bir eğitim sayfası değildir. Kullanıcıların kavramları deneyerek, parametreleri değiştirerek, sonuçları karşılaştırarak ve görevleri tamamlayarak öğrenmesini amaçlayan tarayıcı tabanlı bir yapay zekâ laboratuvarıdır.

## 🇬🇧 English

### About the Project

**AI Voltran** is a browser-based interactive learning experience designed to teach the fundamental concepts of artificial intelligence through practical, engaging and simulation-based activities.

Users train lions representing different artificial intelligence modules, complete laboratory missions, pass pilot examinations and combine all components to form their own AI Voltran.

The project is not a static educational website that only presents information. It is an interactive artificial intelligence laboratory where users learn by experimenting, modifying parameters, comparing results and completing measurable missions.

### Core Modules

* Data and data quality
* Tokenization
* Embeddings and vector search
* Large Language Models
* Transformer architecture
* Prompt engineering
* Context engineering
* RAG
* MCP
* Artificial intelligence agents
* Agent loops
* Loop engineering
* Ethics and safety
* Red teaming
* Human oversight
* Responsible artificial intelligence

### Interactive Laboratories

The project includes interactive experiences supporting the following technologies and artificial intelligence application areas:

* Transformers.js
* TensorFlow.js
* MediaPipe
* WebLLM
* WebGPU
* WebAssembly
* Web Speech API
* Canvas API
* p5.js
* Browser-based agent simulations
* Prompt construction and comparison experiments
* RAG evidence and citation validation
* Neural network visualizations
* Camera and motion detection applications
* Voice command experiments

Large models and external libraries are loaded only when the related application is started.

### Learning Flow

Each module follows this learning process:

1. Discover the concept
2. Explore real-world examples
3. Experiment in an interactive laboratory
4. Complete the required mission conditions
5. Compare and interpret the results
6. Pass the pilot examination
7. Dock the corresponding Voltran component

The final sharing image cannot be unlocked until all required laboratories and safety missions have been completed.

### Language Support

The portal operates synchronously in two languages:

* Turkish
* English

When the language is changed, the main interface, laboratories, missions, status messages, explanations, achievement notifications and sharing areas are updated together.

### Technical Features

* Runs directly in a web browser
* Does not require a backend
* Can be distributed as a single HTML file
* Compatible with desktop and mobile devices
* Progress is stored locally in the browser
* Does not collect personal user data
* Does not require a user account
* Provides Turkish and English support
* Supports keyboard accessibility
* Respects reduced-motion preferences
* Generates social media sharing images

### Usage

Open the HTML file in a modern web browser.

The project should be published over HTTPS for all advanced features to operate correctly. Camera, microphone, WebGPU, model downloads and native sharing features may be restricted by browsers when the project is opened through the `file://` protocol.

To publish through GitHub Pages:

1. Upload the project files to a GitHub repository.
2. Rename the main HTML file to `index.html`.
3. Open **Settings → Pages** in the GitHub repository.
4. Select the branch to publish.
5. Open the project through the generated GitHub Pages URL.
